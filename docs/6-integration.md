## 🤖 Интеграция с Telegram Bot API

### Описание
В проект добавлен Telegram бот для уведомлений клиентов и администратора барбершопа.

### Функциональность
- ✅ Уведомления администратора о новых записях
- ✅ Подтверждения клиентам о созданной записи
- ✅ Напоминания за час до визита
- ✅ Автоматическое сохранение chat_id клиентов

### ИСпользуемый стэк
- Язык: Go
- Библиотека: go-telegram-bot-api
- БД: MySQL (таблица telegram_clients)
