# Технологический стек

## Программное обеспечение для проектирования

- **Figma** - cервис для создания макетов интерфейса, кнопок, страниц сайта, логотипов и иконок
- **Draw.io** - создаёт схемы и диаграммы (ER-диаграммы базы данных, схемы процессов и т.д)

## Програмное обеспечение для разработки

### Бэкенд (серверная часть, логика сайта)

-	**Go (Golang)** — основной язык разработки серверной логики.
-	**Gin / Стандартная библиотека net/http** — обработка HTTP-запросов и маршрутизация.
-	**GORM** — ORM для работы с базой данных (модели, связи, миграции).
-	**go-playground/validator** — валидация входящих данных.


### Фронтэнд

- **HTML** — базовая разметка страниц.
- **CSS** — стилизация элементов (дополнительно к Tailwind).
- **TypeScript** — типизированный JavaScript для надёжности кода.
- **React** — библиотека для построения пользовательского интерфейса.
- **Tailwind CSS** — утилитарный CSS-фреймворк для быстрой стилизации.
- **React Router DOM** — навигация между страницами.
- **Axios** — отправка запросов к бэкенду.

### Базы данных

-	**MySQL** — реляционная база данных (хранение информации о клиентах, мастерах, услугах и записях).
- **Redis** — кеш и очередь задач Celery. (в будущем)
- **S3‑совместимое облако (Yandex Cloud / AWS S3)** - хранение медиа (в будущем)

### Инфраструктура

- **Docker** — контейнеризация проекта (бэкенд, фронтенд, MySQL).
- **Gunicorn + Nginx** — продакшн-сервер.
- **Git + GitHub** — контроль версий и хранение кода.
- **Let’s Encrypt / Certbot** — SSL‑сертификаты для HTTPS.

### Тестирование и отладка

- **Postman** — тестирование API-запросов.
- **Swagger / OpenAPI** — документирование API.

### Аналитика и мониторинг

- **Prometheus** — сбор метрик работы сайта.
- **Grafana** — визуализация метрик.
