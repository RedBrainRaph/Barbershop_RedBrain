## 🔄 Поток данных

### 1. Загрузка страницы Services

```
Пользователь → http://localhost:5173/services
                          ↓
                Services.tsx загружается
                          ↓
                useEffect() запускается
                          ↓
            apiClient.getServices() вызывается
                          ↓
    fetch('http://localhost:8080/api/services')
                          ↓
              Go: getServices() handler
                          ↓
     db.Select(&services, "SELECT * FROM services...")
                          ↓
              MySQL выполняет запрос
                          ↓
            Возвращает массив услуг
                          ↓
          Go отправляет JSON response
                          ↓
       React получает данные и обновляет state
                          ↓
          Услуги отображаются на странице
```

### 2. Создание записи

```
Пользователь → заполняет форму на /booking
                          ↓
                Нажимает "Записаться"
                          ↓
              handleSubmit() срабатывает
                          ↓
         apiClient.createBooking(data)
                          ↓
    fetch('http://localhost:8080/api/bookings', {
        method: 'POST',
        body: JSON.stringify(bookingData)
    })
                          ↓
           Go: createBooking() handler
                          ↓
      Валидация данных (Gin binding)
                          ↓
  Проверка дубликатов времени в БД
                          ↓
     db.Exec("INSERT INTO bookings...")
                          ↓
       MySQL сохраняет запись
                          ↓
      Go возвращает {id, message}
                          ↓
   React показывает экран успеха
```
