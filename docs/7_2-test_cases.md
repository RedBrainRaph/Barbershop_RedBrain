**Заголовок:** Успешная запись на услугу с выбором конкретного барбера

**Предусловие:** Открыта страница онлайн-записи, бэкенд-сервер запущен, в базе данных есть услуги и барберы, выбранное время свободно

| Ввести в поле "Telegram" значение "@ivan_ivanov" | В поле "Telegram" отображается введенное значение |

## Страница отображения услуг

### Просмотр списка услуг

**ID:** TC-007

**Заголовок:** Корректное отображение списка услуг

**Предусловие:** Открыта страница "Услуги", в базе данных есть услуги

| Шаг | Ожидаемый результат |
|-----|---------------------|
| Перейти на страницу "Услуги" через навигационное меню | Открывается страница с заголовком "Наши услуги" |
| Проверить отображение всех услуг из базы данных | Все услуги отображаются на странице |
| Для первой услуги проверить наличие названия | Название отображается корректно |
| Для первой услуги проверить наличие описания | Описание отображается корректно |
| Для первой услуги проверить наличие цены | Цена отображается в формате "1500 ₽" |
| Для первой услуги проверить наличие длительности | Длительность отображается в формате "60 мин" |
| Проверить разделение на категории | Основные услуги отображаются отдельно от дополнительных |

## API-тестирование

### Получение списка услуг через API

**ID:** TC-009

**Заголовок:** Проверка GET-запроса к /api/services через Postman

**Предусловие:** Бэкенд-сервер запущен, Postman открыт

| Шаг | Ожидаемый результат |
|-----|---------------------|
| Создать новый GET-запрос в Postman | Запрос создан |
| Ввести URL: http://localhost:8080/api/services | URL введен |
| Нажать кнопку "Send" | Статус ответа: 200 OK |
| Проверить тело ответа | Ответ содержит JSON-массив с услугами |
| Проверить структуру первого элемента | Элемент содержит поля: id, title, description, price, duration, icon, category |

### Создание записи через API с конфликтом времени

**ID:** TC-010

**Заголовок:** Проверка POST-запроса к /api/bookings с конфликтом времени

**Предусловие:** Бэкенд-сервер запущен, в базе данных уже есть запись для barber_id=1 на 2026-03-15 в 15:00, Postman открыт

| Шаг | Ожидаемый результат |
|-----|---------------------|
| Создать новый POST-запрос в Postman | Запрос создан |
| Ввести URL: http://localhost:8080/api/bookings | URL введен |
| Перейти на вкладку "Body" → "raw" → "JSON" | Настройки тела запроса |
| Ввести JSON:
```json
{
  "client_name": "Тест Тестов",
  "client_phone": "+7 999 888-77-66",
  "client_telegram": "@test",
  "service_id": 1,
  "barber_id": 1,
  "booking_date": "2026-03-15",
  "booking_time": "15:00:00"
}
``` | JSON введен |
| Нажать кнопку "Send" | Статус ответа: 409 Conflict |
| Проверить тело ответа | Ответ содержит сообщение: "Это время уже занято" |
